- Linux, UNIX (require `stty` installed)
- Windows XP or higher (require `HiddenInput.exe`)
- cURL extension
- [SSL Certificate Verification](https://curl.haxx.se/docs/caextract.html)

How to use:

<pre>php getFacebookAndroidAccessToken.php</pre>
